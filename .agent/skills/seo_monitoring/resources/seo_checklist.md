# Detailed SEO Checklist for Daffodil Software Website

## 1. Technical SEO

- [ ] **robots.txt**: Check `https://daffodilsoft.com/robots.txt` for proper indexing directives.
- [ ] **Sitemap**: Verify `https://daffodilsoft.com/sitemap.xml` exists and is updated.
- [ ] **HTTPS**: Ensure all URLs redirect to `https`.
- [ ] **SSL/TLS**: Valid certificates for secure browsing.

## 2. On-Page Structure

- [ ] **Single H1**: Each page has exactly one `<h1>` containing the primary keyword.
- [ ] **Logical Hierarchy**: `<h1>` -> `<h2>` -> `<h3>` flow without skipping levels.
- [ ] **Meta Tags**:
  - Title: `<Primary Keyword> | <Brand Name>` (50-60 chars).
  - Description: Compelling summary with CTA (150-160 chars).
  - OpenGraph (OG): For social sharing (Facebook, LinkedIn).
  - Twitter Cards: For Twitter previews.

## 3. Image Optimization

- [ ] **Alt Tags**: Descriptive text for all images.
- [ ] **File Sizes**: Below 200KB for regular images, optimized for WebP.
- [ ] **Descriptive Names**: `software-development-daffodil.png` instead of `IMG_001.png`.

## 4. Mobile Responsiveness

- [ ] **Viewport Meta**: `<meta name="viewport" content="width=device-width, initial-scale=1">`.
- [ ] **Tap Targets**: Buttons and links are at least 48x48px.
- [ ] **Font Sizes**: Readable (minimum 16px for body text).

## 5. Performance (Core Web Vitals)

- [ ] **LCP (Largest Contentful Paint)**: Under 2.5s.
- [ ] **FID (First Input Delay)**: Under 100ms.
- [ ] **CLS (Cumulative Layout Shift)**: Under 0.1.

## 6. Content Visibility

- [ ] **Lazy Loading**: Use `loading="lazy"` for images and offscreen components.
- [ ] **Preload Font**: Preload critical fonts for faster FCP (First Contentful Paint).
